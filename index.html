LATAR BELAKANG

GAP PENILAIAN RISIKO

BSSN mencatat 241 dugaan insiden kebocoran data sepanjang 2024. Namun, metode penilaian risiko yang ada masih manual, subjektif, dan sulit diakses.

Penelitian ini menjembatani kompleksitas UU PDP 2022 dan NIST CSF 2.0 menjadi alat self-assessment otomatis berbasis web statis.

1. Hybrid Framework
Menggabungkan model kuantitatif Likelihood-Impact dengan aturan kualitatif regulasi.
2. Client-Side Processing
Menerapkan Zero-Knowledge Architecture. Data diproses di browser tanpa ke server.
3. Institutional Trust
Fitur Override otomatis yang menurunkan skor risiko jika layanan terdaftar di OJK/Komdigi.
FITUR UTAMA

MEKANISME ANALISIS

Likelihood-Impact Engine
Algoritma kalkulasi dinamis (R = P x I). Skor dampak naik otomatis untuk data sensitif.
Compliance Mapping
Penerjemah otomatis 17 parameter teknis ke dalam status kepatuhan pasal UU PDP 2022 & NIST CSF 2.0.
Encrypted Persistence
Penyimpanan progres analisis di memori lokal browser menggunakan teknik obfuscation.
