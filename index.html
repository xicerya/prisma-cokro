LATAR BELAKANG

GAP PENILAIAN RISIKO

Seiring meningkatnya transaksi digital, Indonesia menghadapi darurat privasi. BSSN mencatat 241 dugaan kebocoran dengan 56 juta data terekspos sepanjang 2024.

Namun, penilaian risiko saat ini masih manual, subjektif, mahal dan lebih fokus pada keamanan infrastruktur dibanding perlindungan privasi pengguna.

Penelitian ini menyederhanakan kompleksitas UU PDP 2022 dan NIST CSF 2.0 menjadi alat self-assessment otomatis berbasis web statis.

Pendekatan Sistem

PILAR METODOLOGI

1. HYBRID FRAMEWORK
Menggabungkan model kuantitatif Likelihood-Impact dengan aturan kualitatif regulasi.
2. CLIENT-SIDE PROCESSING
Menerapkan Zero-Knowledge Architecture. Data diproses di browser tanpa ke server.
3. INSTITUTIONAL TRUST
Fitur Override otomatis yang menurunkan skor risiko jika layanan terdaftar di OJK/Komdigi PSE/AFPI/Vendor Biometrik.
